# **Slide-Template**

Template for [BTCPay Server](https://btcpayserver.org) presentations, based on [Marp](https://marp.app/).

---

# How to write slides

Split pages by horizontal ruler (`---`). It's very simple! :satisfied:

```markdown
# Intro

foo

---

# Outro

bar
```
